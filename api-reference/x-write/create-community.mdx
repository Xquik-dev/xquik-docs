## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

## Body

<ParamField body="account" type="string" required>
  The connected X account to create the community as. Must be a username you have connected to your Xquik account.
</ParamField>

<ParamField body="name" type="string" required>
  The name for the new community.
</ParamField>

<ParamField body="description" type="string">
  Optional description for the community explaining its purpose.
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="communityId" type="string">The unique ID of the newly created community.</ResponseField>
    <ResponseField name="communityName" type="string">The name of the created community. Omitted if unavailable.</ResponseField>
    <ResponseField name="success" type="boolean">Always `true` on success.</ResponseField>

    ```json
    {
      "communityId": "1893726451023847424",
      "communityName": "My Community",
      "success": true
    }
    ```
  </Tab>
  <Tab title="400 Invalid Input">
    ```json
    { "error": "invalid_input", "message": "Missing or invalid community name or description" }
    ```
    Missing `account`, `name`, or `description`, or the values do not meet validation requirements.
  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated", "message": "Missing or invalid API key" }
    ```
    Missing or invalid API key.
  </Tab>
  <Tab title="402 Subscription Required">
    ```json
    { "error": "no_subscription", "message": "No active subscription" }
    ```
    An active subscription is required to create communities. Subscribe from the [dashboard](https://xquik.com/subscription).
  </Tab>
  <Tab title="402 Usage Limit">
    ```json
    { "error": "usage_limit_reached", "message": "Monthly usage quota exceeded" }
    ```

    Your monthly usage quota has been exhausted. Usage resets at the start of each billing cycle.
  </Tab>
  <Tab title="404 Account Not Found">
    ```json
    { "error": "account_not_found", "message": "Connected X account not found" }
    ```
    The specified `account` is not connected to your Xquik account. Connect it from the [dashboard](https://xquik.com/dashboard).
  </Tab>
</Tabs>
