---
title: "Delete Webhook"
description: "Permanently remove a webhook endpoint and stop all future deliveries."
api: "DELETE /webhooks/{id}"
keywords: ["webhook", "delete", "remove", "deactivate"]
---

<Info>
  This endpoint does not count against your monthly usage quota.
</Info>

## Path Parameters

<ParamField path="id" type="string" required>
  The webhook ID to delete.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. This endpoint also accepts session cookie authentication.
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="success" type="boolean">
      Always `true` on successful deletion.
    </ResponseField>

    ```json
    { "success": true }
    ```

    The webhook is deactivated immediately. Pending deliveries that are already in-flight may still be attempted, but no new deliveries will be queued.
  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated" }
    ```
    Missing or invalid API key / session cookie.
  </Tab>
  <Tab title="400 Invalid ID">
    ```json
    { "error": "invalid_id" }
    ```
    The provided webhook ID is not a valid format.
  </Tab>
  <Tab title="404 Not Found">
    ```json
    { "error": "not_found" }
    ```
    No webhook exists with this ID, or it belongs to a different account.
  </Tab>
</Tabs>

<Note>
  This endpoint supports **dual authentication** -- either API key (`x-api-key` header) or session cookie from the dashboard.

</Note>
