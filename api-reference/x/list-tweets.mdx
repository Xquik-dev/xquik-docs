  ```bash cURL
  curl "https://xquik.com/api/v1/x/lists/1234567890/tweets" \
    -H "x-api-key: xq_YOUR_KEY_HERE" | jq
  ```

  ```javascript Node.js
  const listId = "1234567890";
  const response = await fetch(`https://xquik.com/api/v1/x/lists/${listId}/tweets`, {
    headers: { "x-api-key": "xq_YOUR_KEY_HERE" },
  });
  const data = await response.json();
  console.log(data);
  ```

  ```python Python
  import requests

  response = requests.get(
      "https://xquik.com/api/v1/x/lists/1234567890/tweets",
      headers={"x-api-key": "xq_YOUR_KEY_HERE"},
  )
  data = response.json()
  print(data)
  ```

## Path parameters

<ParamField path="id" type="string" required>
  List ID (numeric string).
</ParamField>

## Query parameters

<ParamField query="cursor" type="string">
  Pagination cursor from a previous response. Omit for the first page.
</ParamField>

<ParamField query="sinceTime" type="string">
  ISO 8601 timestamp. Only return tweets after this time.
</ParamField>

<ParamField query="untilTime" type="string">
  ISO 8601 timestamp. Only return tweets before this time.
</ParamField>

<ParamField query="includeReplies" type="boolean">
  Include reply tweets. Default: `false`.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported.
</ParamField>
