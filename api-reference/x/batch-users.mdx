
  ```javascript Node.js
  const ids = "44196397,987654321";
  const response = await fetch(`https://xquik.com/api/v1/x/users/batch?ids=${ids}`, {
    headers: { "x-api-key": "xq_YOUR_KEY_HERE" },
  });
  const data = await response.json();
  console.log(data);
  ```

  ```python Python
  import requests

  response = requests.get(
      "https://xquik.com/api/v1/x/users/batch",
      params={"ids": "44196397,987654321"},
      headers={"x-api-key": "xq_YOUR_KEY_HERE"},
  )
  data = response.json()
  print(data)
  ```
</CodeGroup>

## Query parameters

<ParamField query="ids" type="string" required>
  Comma-separated user IDs. Maximum 100 per request.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported.
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="users" type="object[]">Array of user profiles matching the requested IDs.</ResponseField>
    <ResponseField name="has_next_page" type="boolean">Always `false` for batch requests.</ResponseField>
    <ResponseField name="next_cursor" type="string">Always empty for batch requests.</ResponseField>

    ```json
    {
      "users": [
        {
          "id": "44196397",
          "username": "elonmusk",
          "name": "Elon Musk",
          "followers": 200000000,
          "verified": true
        }
      ],
      "has_next_page": false,
      "next_cursor": ""
    }
    ```
  </Tab>
  <Tab title="400 Missing IDs">

    ```json
    { "error": "missing_ids", "message": "ids parameter required" }
    ```
  </Tab>
  <Tab title="400 Too many IDs">

  </Tab>
  <Tab title="401 Unauthenticated">
