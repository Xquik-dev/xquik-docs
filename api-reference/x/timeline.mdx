---
title: "Get timeline"
description: "Retrieve the home timeline from the authenticated X account with optional seen-tweet filtering"
api: "GET /x/timeline"
keywords: ["timeline", "home feed", "x api"]
---

<Info>
  **Cost:** 1 credit per result returned (~$0.00015 on [PAYG](/guides/billing#pay-per-use-mpp)).
</Info>

<Note>
  Requires a connected X account. Uses user-authenticated access.
</Note>

<CodeGroup>
  ```bash cURL
  curl https://xquik.com/api/v1/x/timeline \
    -H "x-api-key: xq_YOUR_KEY_HERE" | jq
  ```

  ```javascript Node.js
  const response = await fetch("https://xquik.com/api/v1/x/timeline", {
    headers: { "x-api-key": "xq_YOUR_KEY_HERE" },
  });
  const data = await response.json();
  console.log(data);
  ```

  ```python Python
  import requests

  response = requests.get(
      "https://xquik.com/api/v1/x/timeline",
      headers={"x-api-key": "xq_YOUR_KEY_HERE"},
  )
  data = response.json()
  print(data)
  ```
</CodeGroup>

## Query parameters

<ParamField query="seenTweetIds" type="string">
  Comma-separated tweet IDs to exclude from results. Use this to avoid returning tweets the user has already seen.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported.
</ParamField>

## Response

<Tabs>

    <ResponseField name="tweets" type="object[]">
