## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported.
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

## Body

Provide either `tweetId` or `tweetUrl`. At least 1 is required.

<ParamField body="tweetId" type="string">
  Numeric tweet ID (e.g. `"1893456789012345678"`).
</ParamField>

<ParamField body="tweetUrl" type="string">
  Full tweet URL. Accepts `x.com` and `twitter.com` formats (e.g. `https://x.com/user/status/1893456789012345678`).
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="tweetId" type="string">
      Resolved tweet ID.
    </ResponseField>
    <ResponseField name="media" type="object[]">
      Array of downloaded media items.
      <Expandable title="media object">
        <ResponseField name="url" type="string">Permanent download URL hosted on `media.xquik.com`.</ResponseField>
        <ResponseField name="type" type="string">Media type: `photo`, `video`, or `animated_gif`.</ResponseField>
        <ResponseField name="index" type="number">Position in the tweet's media attachments (0-indexed).</ResponseField>
        <ResponseField name="fileSize" type="string">File size in bytes (as a string). Omitted if unavailable.</ResponseField>
      </Expandable>
    </ResponseField>

    ```json
    {
      "tweetId": "1893456789012345678",
      "media": [
        {
          "url": "https://media.xquik.com/dl/abc123/video.mp4",
          "type": "video",
          "index": 0,
          "fileSize": "4821000"
        },
        {
          "url": "https://media.xquik.com/dl/def456/photo.jpg",
          "type": "photo",
          "index": 1,
          "fileSize": "245000"
        }
      ]
    }
    ```

    <Info>
      First download is metered and counts toward your monthly usage quota. Subsequent requests for the same tweet return cached URLs at no cost. Downloads are saved to your gallery at `https://xquik.com/gallery`.
    </Info>
  </Tab>
  <Tab title="400 Invalid Input">
    ```json
    { "error": "invalid_input", "message": "Invalid request body" }
    ```
    Missing or malformed request body.
  </Tab>
  <Tab title="400 Invalid Tweet ID">

    The provided tweet ID or URL could not be resolved to a valid tweet ID.
  </Tab>


  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated", "message": "Missing or invalid API key" }
    ```
