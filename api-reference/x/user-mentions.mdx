## Path parameters

<ParamField path="id" type="string" required>
  User ID (numeric) or username.
</ParamField>

## Query parameters

<ParamField query="cursor" type="string">
  Pagination cursor from a previous response. Omit for the first page.
</ParamField>

<ParamField query="sinceTime" type="string">
  ISO 8601 timestamp. Only return mentions after this time.
</ParamField>

<ParamField query="untilTime" type="string">
  ISO 8601 timestamp. Only return mentions before this time.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported.
</ParamField>
