---
title: "Delete Integration"
description: "Remove an integration and stop receiving notifications"
api: "DELETE /integrations/{id}"
keywords: ["integration", "delete", "remove"]
---

<Info>
  This endpoint does not count against your monthly usage quota.
</Info>

<CodeGroup>
  ```bash cURL
  curl -X DELETE https://xquik.com/api/v1/integrations/4 \
    -H "x-api-key: xq_YOUR_KEY_HERE" | jq
  ```

  ```javascript Node.js
  const integrationId = "4";

    },
  });
  const data = await response.json();
  ```

  ```python Python

  integration_id = "4"

  )
  data = response.json()
  ```

## Path parameters

<ParamField path="id" type="string" required>
  The unique integration ID.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="success" type="boolean">Always `true` on successful deletion.</ResponseField>

    ```json
    { "success": true }
    ```
  </Tab>
  <Tab title="400 Invalid ID">
    ```json
    { "error": "invalid_id", "message": "Invalid integration ID format" }
    ```
    The provided integration ID is not a valid format.
  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated", "message": "Missing or invalid API key" }
    ```
    Missing or invalid API key.
  </Tab>
  <Tab title="404 Not Found">
    ```json
    { "error": "not_found", "message": "Integration not found" }
    ```
    No integration exists with this ID, or it belongs to a different account.
  </Tab>
</Tabs>

<Info>
  The integration is permanently deleted and no further notifications will be sent. Delivery history is retained.
</Info>

<Note>
  **Related:** [List Integrations](/api-reference/integrations/list) to verify the integration was removed, or [Create Integration](/api-reference/integrations/create) to set up a new one.
</Note>
