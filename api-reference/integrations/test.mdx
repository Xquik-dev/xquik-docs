      var data map[string]interface{}

## Path parameters

<ParamField path="id" type="string" required>
  The unique integration ID.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="success" type="boolean">Always `true` when the test notification was sent successfully.</ResponseField>

    ```json
    { "success": true }
    ```
  </Tab>
  <Tab title="400 Invalid ID">
    ```json
    { "error": "invalid_id", "message": "Invalid integration ID format" }
    ```
    The provided integration ID is not a valid format.
  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated", "message": "Missing or invalid API key" }
    ```
    Missing or invalid API key.
  </Tab>
  <Tab title="404 Not Found">
    ```json
    { "error": "not_found", "message": "Integration not found" }
    ```
    No integration exists with this ID, or it belongs to a different account.
  </Tab>
  <Tab title="502 Delivery Failed">
    ```json
    { "error": "delivery_failed", "message": "Test notification could not be delivered" }
    ```
    The test notification failed to deliver. Check the integration configuration (e.g. Telegram chat ID, bot permissions).
  </Tab>
</Tabs>

<Info>
  A test notification sends a sample event to verify connectivity. It does not create a real event or appear in delivery history.
</Info>
