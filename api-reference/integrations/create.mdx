## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

## Body

<ParamField body="type" type="string" required>
  Integration type. Currently supported: `"telegram"`.
</ParamField>

<ParamField body="name" type="string" required>
  A human-readable name for the integration.
</ParamField>

<ParamField body="config" type="object" required>
  Configuration object for the integration type.
  <Expandable title="config for telegram">
    <ParamField body="config.chatId" type="string" required>
      Telegram chat ID to send notifications to. Use a negative number for group chats.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="eventTypes" type="string[]" required>
  Array of event types to subscribe to. At least 1 required. Valid types: `tweet.new`, `tweet.quote`, `tweet.reply`, `tweet.retweet`, `draw.completed`, `extraction.completed`, `extraction.failed`.
</ParamField>

## Response

<Tabs>
