## Path Parameters

<ParamField path="slug" type="string" required>
  The flow slug.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

## Body

<ParamField body="stepType" type="string" required>
  One of: `action`, `condition`, `extraction`.
</ParamField>

<ParamField body="branch" type="string" required>
  One of: `main`, `if_true`, `if_false`. Use `main` for linear flows. Use `if_true` or `if_false` for condition branches.
</ParamField>

<ParamField body="config" type="object" required>
  Step-specific configuration. Contents vary by step type & action/extraction type.
</ParamField>

<ParamField body="position" type="integer">
  Order position within the branch. Steps execute in ascending order.
</ParamField>

<ParamField body="parentStepId" type="string">
  Parent step ID. Required for `if_true` & `if_false` branches to identify the condition step.
</ParamField>

<ParamField body="actionType" type="string">
  Required when `stepType` is `action`. One of: `create_tweet`, `follow`, `like`, `reply_tweet`, `retweet`, `send_dm`, `send_email`, `send_telegram`, `unfollow`.
</ParamField>

<ParamField body="extractionType" type="string">
  Required when `stepType` is `extraction`. One of the 20 extraction types (e.g., `followers`, `following`, `tweet_replies`).
</ParamField>

<ParamField body="outputName" type="string">
  Required when `stepType` is `extraction`. Variable name for the extraction output. Alphanumeric characters only, max 64 characters. Reference this in later steps.
</ParamField>
