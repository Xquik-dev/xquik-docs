## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

## Body

<ParamField body="name" type="string" required>
  Flow name. Used to generate the URL slug.
</ParamField>

<ParamField body="triggerType" type="string" required>
  One of: `monitor_event`, `schedule`, `search`, `webhook_inbound`.
</ParamField>

<ParamField body="triggerConfig" type="object" required>
  Trigger-specific configuration object. Contents vary by `triggerType`.
</ParamField>

<ParamField body="templateSlug" type="string">
  Template slug to scaffold the flow from. Omit to start from scratch.
</ParamField>
