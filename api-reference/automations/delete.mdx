---

api: "DELETE /automations/{slug}"
keywords: ["automation", "flow", "delete", "remove"]
---

<CodeGroup>
  ```bash cURL
  curl -X DELETE https://xquik.com/api/v1/automations/new-follower-welcome \
    -H "x-api-key: xq_YOUR_KEY_HERE" | jq
  ```

  ```javascript Node.js
  const slug = "new-follower-welcome";
  const response = await fetch(`https://xquik.com/api/v1/automations/${slug}`, {
    method: "DELETE",
    headers: {
      "x-api-key": "xq_YOUR_KEY_HERE",
    },
  });
  const data = await response.json();
  ```

  ```python Python
  import requests

  slug = "new-follower-welcome"
  response = requests.delete(
      f"https://xquik.com/api/v1/automations/{slug}",
      headers={"x-api-key": "xq_YOUR_KEY_HERE"},
  )
  data = response.json()
  ```

  ```go Go
  package main
  ```

## Path Parameters

<ParamField path="slug" type="string" required>
  The flow slug.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

## Response

<Tabs>
  <Tab title="200 OK">
    <ResponseField name="success" type="boolean">Always `true` on successful deletion.</ResponseField>

    ```json
    { "success": true }
    ```
  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated", "message": "Missing or invalid API key" }
    ```
    Missing or invalid API key.
  </Tab>
  <Tab title="404 Not Found">
    ```json
    { "error": "not_found", "message": "Flow not found" }
    ```
    No flow exists with this slug, or it belongs to a different account.
  </Tab>
</Tabs>

<Info>
  Deleting a flow removes all its steps and run history. This action is irreversible.
</Info>

<Note>
  **Related:** [List Automations](/api-reference/automations/list) to verify the flow was removed, or [Create Automation](/api-reference/automations/create) to set up a new one.
</Note>
