## Path Parameters

<ParamField path="slug" type="string" required>
  The flow slug.
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key. Session cookie authentication is also supported. Generate a key from the [dashboard](https://xquik.com/dashboard).
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

## Body

<ParamField body="stepId" type="string" required>
  The unique step ID to update.
</ParamField>

<ParamField body="stepType" type="string">
  One of: `action`, `condition`, `extraction`.
</ParamField>

<ParamField body="branch" type="string">
  One of: `main`, `if_true`, `if_false`.
</ParamField>

<ParamField body="config" type="object">
  Updated step configuration.
</ParamField>

<ParamField body="position" type="integer">
  Updated order position within the branch.
</ParamField>

<ParamField body="parentStepId" type="string">
  Updated parent step ID for branching.
</ParamField>

<ParamField body="actionType" type="string">
  Updated action type. One of: `create_tweet`, `follow`, `like`, `reply_tweet`, `retweet`, `send_dm`, `send_email`, `send_telegram`, `unfollow`.
</ParamField>

<ParamField body="extractionType" type="string">
  Updated extraction type.
</ParamField>

<ParamField body="outputName" type="string">
  Updated extraction output variable name. Must start with a letter, then letters, digits, or underscores. Max 64 characters. Reserved names: `previous`, `username`, `tweetId`, `tweet`, `userId`, `cron`, `trends`, `query`, `results`.
</ParamField>
