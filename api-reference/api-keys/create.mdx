---
title: Create API Key
description: Generate a new API key for programmatic access
---

<Note>
  This endpoint requires **session cookie** authentication only (dashboard).
</Note>

## Request

```
POST /api/v1/api-keys
```

### Body

```json
{
  "name": "Production"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | No | Display name (defaults to `"Default"`) |

## Response

<Tabs>
  <Tab title="201 Created">
    ```json
    {
      "id": "42",
      "fullKey": "xqk_a1b2c3d4e5f6789012345678901234",
      "prefix": "a1b2",
      "name": "Production",
      "createdAt": "2026-02-24T10:30:00.000Z"
    }
    ```

    <Warning>
      The `fullKey` is returned **only once**. Store it securely. It cannot be retrieved again.
    </Warning>
  </Tab>
  <Tab title="401 Unauthenticated">
    ```json
    { "error": "unauthenticated" }
    ```
  </Tab>
</Tabs>
