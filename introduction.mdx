## For Developers

Xquik provides 4 integration methods:

<Steps>
  <Step title="REST API v1">
    Create monitors, query events, and manage webhooks programmatically. Authenticate with API keys via the `x-api-key` header.
  </Step>
  <Step title="HMAC Webhooks">
    Receive real-time event notifications at your HTTPS endpoints. Every delivery is signed with SHA-256 HMAC for verification.
  </Step>
  <Step title="MCP Server">
    Connect AI agents (Claude, ChatGPT, Grok) to Xquik via the Model Context Protocol. 37 tools available at `xquik.com/mcp`.
  </Step>
  <Step title="AI Agent Skill">
    Install the [Xquik Skill](https://github.com/Xquik-dev/x-twitter-scraper) to give 40+ AI coding agents (Claude Code, Cursor, Codex, Copilot, Windsurf, and more) deep knowledge of the Xquik API. Install via `npx skills add Xquik-dev/x-twitter-scraper`.
  </Step>
</Steps>
